RED SCRABBLE

## Introduction
Welcome to the Scrabble Game! This is a simple Python-based implementation of the classic Scrabble board game, with a countdown spin on it.
I started this game in July 2023 and have completed it in March 2024 after a 6 month hiatus.

## Features
- Single player.
- Real-time scoring.
- Offline tracking
- Timer to add some pressure.
- Ability to shuffle tiles ONCE and double your score ONCE.

## Requirements
- Python 3.x installed on your system.
- Pygame library for timings.

## Installation
1. Clone or download the repository from [GitHub](https://github.com/Mimadiyiz/Red Scrabble V2).
2. Navigate to the project directory.

## Usage
1. Open the file and run the executable

## How to Play
- Use your mouse to interact with the game interface.
- Type a valid English word using the letters on your rack.
- Can use letters more than once.
-Only 30 seconds on the timer.
- Click the "Submit" button to submit your word.
- Use the "Shuffle" button to shuffle your tiles ~ ONE per game.
- Use the "Double Score" button to double your score for the next word ~ ONE per game.
